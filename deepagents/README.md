## Key Findings

### What's Included in deepagents

1. **Planning** — `TodoListMiddleware` for task breakdown and progress tracking
2. **Filesystem** — `ls`, `read_file`, `write_file`, `edit_file`, `glob`, `grep` tools
3. **Shell access** — `execute` tool (requires sandbox backend)
4. **Sub-agents** — `task` tool for spawning isolated context windows
5. **Smart defaults** — Opinionated prompts that teach tool usage
6. **Context management** — Auto-summarization when conversations get long

### Architecture

deepagents is built on:
- **LangGraph** — Runtime with checkpointers, streaming, persistence
- **LangChain** — Agent framework with middleware pattern
- **Middleware Stack** — Composable layers (each capability = 1 middleware)

### Orchestrator Assessment

lilCel (OpenClaw) is ~85% equivalent to deepagents out of the box. Missing:
1. Formal `write_todos` task tracking (we use notes instead)
2. Auto-summarization middleware (we use compaction)
3. ACP sub-agent configuration
